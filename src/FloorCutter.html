Dan's Floor Cutting Guide

Hi! I wanted to make this quick website so that anyone can access my quick math formulas to quickly get a range of numbers to use for starting boards that

  • 1) Will not create slivers at the end of a row
  • 2) Will give a range of values to cut from so that board may be staggered to your liking
  • 3) Will give 2 ranges of values based on the number of boards that can be fit. Generally there is a range with more boards and shorter cuts, and less boards with bigger cuts
  • 4) Will give the user flexibility in the accepted range of shortest and longest board cut accepted
  • 5) Will automatically subtract the gap size, which can also be specified, to be subtracted from both sides of the length given and a new actual length displayed

Additionally, I will include a quick floor width guide. This will include some of the same features
BUT NOT the floor width but only for the last board, because the walls cannot always be trusted
just trust the 1/2 mark
I'd like the user to know how many boards will be used given a floor width input for both
a board seam aligning with the center mark of the floor, and a board's center aligning with the floor center mark
both will give a measurement to the halfway mark, but the latter should give the measurement to the mark of the end of a board. this can be done with width/2-width of board/2
the halfway mark should still be make, and I'll include a mark from the halfway mark to the first row to make and lock in, so the 2nd row... if you're not a computer scientist
I'll also make a measurement off the wall to this mark for users if they prefer... Generally both marks should be make to ensure that your measurements were correct, so use that part as a sanity check :)
